# 업무-현황

## 문서 목적

이 문서는 지금까지 나온 결과물과 진척 상태를 기록하는 증거 문서입니다. AI가 비전 대비 어디까지 왔는지 판단하고 다음 아젠다/Todo/Task를 만드는 근거가 됩니다.

## 이 문서가 답해야 할 질문

- 현재 실제로 존재하는 결과물은 무엇인가?
- 무엇이 완료되었고, 무엇이 비어 있는가?
- 테스트, 운영, 사용자 반응, 사업 지표 중 어떤 근거가 있는가?

## AI 행동 규칙

- 추정이 아니라 증거 위주로 적는다.
- 완료라고 적었으면 검증 근거를 같이 남긴다.
- 생산 이후에는 현황을 갱신해 다음 루프의 입력으로 만든다.

## 작성 템플릿

### 1. 현재 결과물
- 앱/기능 상태
- 문서/콘텐츠 상태
- 운영/사업 결과물 상태

### 2. 완료 영역
- 완료된 것
- 완료 근거

### 3. 미완료 영역
- 비어 있는 것
- 막힌 이유

### 4. 검증 / 증거
- 테스트 결과
- 운영 로그
- 사용자 반응
- 수치/지표

### 5. 비전 대비 판단
- 어디까지 왔는가
- 다음에 메워야 할 차이
- 연결 아젠다 후보
