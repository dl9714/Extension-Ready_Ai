# 업무-비전

## 문서 목적

이 문서는 사용자가 이 프로젝트를 왜 하는지, 어디까지 가고 싶은지, 무엇을 절대 해치면 안 되는지 정의하는 최상위 방향 문서입니다. AI는 작업을 시작하기 전에 이 문서를 먼저 읽고, 이후 모든 기획과 생산이 이 비전을 벗어나지 않는지 계속 확인해야 합니다.

## 이 문서가 답해야 할 질문

- 왜 이 앱/콘텐츠/운영체계를 만드는가?
- 누구를 위한 결과물인가?
- 성공했다고 말할 수 있는 기준은 무엇인가?
- 절대 훼손하면 안 되는 가치와 금지사항은 무엇인가?
- 장기적으로 어떤 사업/자산/브랜드로 자라나길 원하는가?

## AI 행동 규칙

- 비전 없이 Task만 보고 바로 생산하지 않는다.
- 비전과 충돌하는 아이디어는 Todo로 올리기 전에 먼저 재검토한다.
- 작업 중 방향이 흔들리면 이 문서로 돌아와 기준을 다시 맞춘다.
- 추측으로 비전을 채우지 말고, 모르는 것은 아젠다의 질문으로 올린다.

## 작성 템플릿

### 1. 존재 이유
- 이 프로젝트를 시작한 이유
- 해결하려는 문제
- 만들고 싶은 변화

### 2. 목표 사용자 / 대상
- 핵심 사용자
- 보조 사용자
- 이들이 얻어야 하는 가치

### 3. 성공 기준
- 기능적 완료 기준
- 운영적 완료 기준
- 사업적/성장 기준

### 4. 금지사항
- 하지 말아야 할 방식
- 절대 허용하지 않을 품질 저하
- 사용자 경험/운영 상 금지선

### 5. 장기 방향
- 나중에 어떤 자산으로 발전하길 원하는가
- 앱 외에 문서/콘텐츠/운영체계로 어떻게 확장될 수 있는가
