# 업무-Task

## 문서 목적

이 문서는 Todo를 실제 실행 가능한 최소 단위로 쪼갠 작업 문서입니다. 코딩, 문서 작성, 검증, 배포, 운영 처리 등 실제 생산은 결국 Task 단위로 진행됩니다.

## 이 문서가 답해야 할 질문

- 지금 바로 실행할 수 있는 최소 단위는 무엇인가?
- 산출물은 무엇이며, 어떻게 검증할 것인가?
- 끝났다고 판단하는 체크 조건은 무엇인가?

## AI 행동 규칙

- Task는 모호하면 안 된다. 바로 손을 댈 수 있어야 한다.
- Task마다 기대 산출물과 검증 방법을 남긴다.
- 끝난 Task는 결과와 함께 현황으로 환류한다.
- 비전과 무관한 Task 증식은 경계한다.
