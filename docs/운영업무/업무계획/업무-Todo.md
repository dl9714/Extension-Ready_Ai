# 업무-Todo

## 문서 목적

이 문서는 하기로 확정한 작업 묶음을 관리하는 실행 계획 문서입니다. 아젠다에서 정리된 논점 중 지금 실제로 추진할 것만 Todo에 올립니다.

## 이 문서가 답해야 할 질문

- 지금 실제로 진행할 작업 묶음은 무엇인가?
- 각 Todo의 완료 조건은 무엇인가?
- 어떤 선행 조건과 우선순위를 가지는가?

## AI 행동 규칙

- 후보나 아이디어를 바로 Todo로 올리지 않는다. 확정된 것만 담는다.
- 너무 큰 Todo는 Task로 내려가기 전에 한 번 더 쪼갠다.
- 완료된 Todo는 현황에 반영하고, 오래된 미진행 Todo는 재평가한다.

## 작성 템플릿

### Todo 항목
- 제목:
- 왜 지금 하는가:
- 완료 조건:
- 선행 의존성:
- 우선순위:
- 연결된 비전/아젠다:
- 주관점:
- 보조관점:

### 메모
- Task로 내려가기 전 주의사항
- 관련 링크/문서
