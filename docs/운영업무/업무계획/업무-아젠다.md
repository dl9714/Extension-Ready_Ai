# 업무-아젠다

## 문서 목적

이 문서는 지금 당장 무엇을 논의하고 판단해야 하는지 정리하는 운영 논점 문서입니다. 비전이 장기 방향이라면, 아젠다는 현재 시점의 초점과 의사결정 목록입니다.

## 이 문서가 답해야 할 질문

- 지금 가장 먼저 다뤄야 할 핵심 논점은 무엇인가?
- 무엇이 막혀 있고, 어떤 결정이 필요하며, 무엇을 유예할 것인가?
- 비전 대비 현재 상태에서 가장 중요한 차이는 무엇인가?

## AI 행동 규칙

- 아젠다는 Todo보다 앞선다. 방향이 정리되지 않았으면 Todo를 늘리지 않는다.
- 논점과 실행을 섞지 않는다. 아직 결정이 필요한 것은 아젠다에 둔다.
- 해결된 논점은 Todo나 현황으로 이동시키고, 오래된 아젠다는 정리한다.

## 작성 템플릿

### 1. 이번 사이클 핵심 논점
- 지금 가장 중요한 주제
- 왜 지금 중요한가

### 2. 결정 필요 항목
- 결정을 내려야 하는 것
- 판단 기준
- 결정 지연 시 영향

### 3. 막힘 / 리스크
- 진행을 막는 요소
- 해결 후보
- 도움 필요 여부

### 4. 보류 / 후보 메모
- 아직 Todo로 올리지 않은 생각
- 백로그와 연결되는 아이템
